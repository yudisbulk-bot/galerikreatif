# Galeri Kreatif

Web galeri karya berbasis Firebase.  
Fitur utama: publikasi gambar (upload atau drawing), judul + deskripsi, ruang koleksi.

## Cara jalan
- Buka `index.html` untuk publik.
- Buka `admin.html` untuk panel admin (password: `admin234`).
- Pastikan koneksi internet aktif (Firebase Storage).

## Tech stack
- Firebase (Firestore + Storage)
- HTML/CSS/JS vanilla
